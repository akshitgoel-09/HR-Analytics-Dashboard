# 📊 HR Analytics Dashboard (Power BI)

## 📌 Overview
This project is an **HR Analytics Dashboard** built using **Power BI** to analyze employee data and uncover insights related to attrition, workforce demographics, and job satisfaction.

The dashboard helps HR teams and decision-makers understand trends, identify risks, and make data-driven decisions.

---

## 🚀 Features

- 👥 **Employee Overview**
  - Total Employees
  - Active Employees
  - Attrition Count & Rate
  - Average Age

- 📉 **Attrition Analysis**
  - Department-wise attrition
  - Education field-wise attrition
  - Age group attrition trends

- 📊 **Demographic Insights**
  - Employee distribution by age group
  - Gender distribution

- 😊 **Job Satisfaction Analysis**
  - Satisfaction levels by job roles
  - Comparative insights across roles

---

## 🛠️ Tools & Technologies

- Power BI Desktop
- Data Modeling (Relationships, Measures)
- DAX (Data Analysis Expressions)
- Data Visualization
